# Web Development Practice Projects

This repository contains a collection of frontend web development practice projects. These projects are built using foundational and modern web technologies including **HTML**, **CSS**, **JavaScript**, and **React**. Each project serves a different purpose and helps reinforce core skills in layout, styling, user interaction, and responsive design.

## 📁 Contents

### 1. Portfolio Landing Page
A minimal personal portfolio landing page designed to showcase basic HTML and CSS layout structure. This project focuses on visual presentation and clean design suitable for personal branding or resume-style websites.

### 2. Library Information Landing Page
An informative landing page created for a fictional library. It demonstrates structured content layout, accessibility considerations, and styling that enhances readability and clarity.

### 3. Coffee Shop Landing Page
A stylized landing page for a coffee shop. This project includes a **functional booking/reservation form**, styled inputs, and user interaction features. It reflects real-world UI elements and use of modern CSS techniques.

## 🛠️ Technologies Used

- HTML5  
- CSS3 (including Flexbox and Grid)  
- JavaScript (vanilla)  
- React (in selected components/projects)  
- Responsive design techniques  

## 📌 Purpose

The goal of this repository is to:
- Practice frontend development fundamentals
- Build fully functional, visually appealing landing pages
- Understand layout techniques and responsive design
- Explore real-world application structures with interactive forms
